# Exploratory Analysis

This folder contains exploratory analysis scripts and generated plots for investigating experiment results.

## Structure

- `scripts/` - Python scripts for generating plots and analysis
- `plots/` - Generated visualization files (PNG images, JSON analysis files)

## Naming Convention

All files use self-explanatory names:
- Barplots: `accuracy_comparison_*.png`
- Venn diagrams: `venn_diagram_{dataset}_local_local_cot_remote.png`
- Analysis JSONs: `venn_analysis_{dataset}_local_local_cot_remote.json`
- Scripts: `plot_{description}_{type}.py`
